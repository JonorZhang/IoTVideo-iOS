+ getPlaybackListV2OfDevice:pageIndex:countPerPage:startTime:endTime:filterType:completionHandler:

获取一页回放文件列表

+ (void)getPlaybackListV2OfDevice:(NSString *)deviceId pageIndex:(uint32_t)pageIndex countPerPage:(uint32_t)countPerPage startTime:(NSTimeInterval)startTime endTime:(NSTimeInterval)endTime filterType:(nullable NSString *)filterType completionHandler:(PlaybackListCallback)completionHandler

Parameters

deviceId

设备ID
pageIndex

页码索引，获取指定页码的回放文件（ pageIndex从0开始递增）
countPerPage

在[startTime, endTime]时间范围内按每页countPerPage个文件查询，每次返回一页的数据，该值由APP设置，取值范围[1, 3200]
startTime

开始时间戳（秒）
endTime

结束时间戳（秒）
filterType

过滤文件类型, nil为不过滤
completionHandler

结果回调

Discussion

Note: ⚠️请根据实际情况合理设置查询时间范围和分页，时间跨度太长或每页数量过大会增加设备查询时间， 建议[startTime, endTime]区间不超过24小时 或 countPerPage不超过1440个文件.
